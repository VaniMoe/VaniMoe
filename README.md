## 🎮 Featured Projects

Hier sind meine beiden Hauptprojekte, an denen ich aktuell arbeite:

### 🌟 [PokeRouge-UnlockAll](https://github.com/VaniMoe/PokeRouge-UnlockAll)
Ein praktisches Web-Tool für das beliebte Browserspiel **PokéRouge**, mit dem sich alle Inhalte, Pokémon und Freischaltungen unkompliziert verwalten und aktivieren lassen.
* **Tech Stack:** HTML, CSS, JavaScript

### 🛠️ [PokeLike Save Editor Overlay](https://github.com/VaniMoe/PokeLike-Save-Editor-Overlay)
Eine komfortable Chrome-Erweiterung für das Roguelike **PokéLike**. Sie injiziert ein In-Game-Overlay, mit dem du dein Team, deine Stats, deinen Pokédex und deine Pokédollar direkt im laufenden Spiel manipulieren kannst, geschützt durch einen automatischen Anti-Erkennungs-Bypass.
* **Tech Stack:** Chrome Extension Manifest V3, JavaScript, PokéAPI
